I'm a developer currently on a journey to master Frontend Web Development. This project is one of my first deep dives into building a functional, real-world application using Vanilla JavaScript.

💡 What I Learned from Scrimba

This project was a huge stepping stone for me. Beyond just writing code, I learned how to think like a developer to solve specific UI and logic problems:
State Management: I learned how to use an array (itemsInCart) to keep track of user selections and sync that "state" with the UI every time an item is added or removed.
The "Magic" of Centering: I mastered the CSS "Center Formula" using position: fixed, top: 50%, left: 50%, and transform: translate(-50%, -50%) to ensure modals look perfect on any screen size.
Event Delegation: Instead of adding an event listener to every single button, I learned how to use a single listener on the document to catch clicks using dataset attributes. This makes the code much cleaner and more efficient.
Semantic HTML & Forms: I moved from using simple div tags to using a proper <form> with <input> types. This taught me about built-in browser validation (like required and minlength) and how to use e.preventDefault() to handle data without refreshing the page.
Data Structures: I practiced using JavaScript array methods like .find() to locate specific items in a data file and .splice() to precisely remove items from the cart.
